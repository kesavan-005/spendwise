# SpendWise

SpendWise is a personal expense tracking web app built using **React + Firebase Firestore**.

## Features
- Add debit / credit / cash transactions
- Live balance & summary
- Category management
- Reports with graphs
- PDF export
- Light/Dark theme
- PWA support (Installable app)

## Tech Stack
- React + Vite
- TailwindCSS
- Firebase Firestore
- Recharts
- jsPDF
